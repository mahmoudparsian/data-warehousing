# Slide 17 --- Flights Case Study: Reliability Ranking

``` sql
WITH carrier_perf AS (
  SELECT carrier_code,
         COUNT(*) flights,
         AVG(dep_delayed_15) delay_rate
  FROM flight_gold
  WHERE flight_status='Completed'
  GROUP BY carrier_code
)
SELECT *
FROM carrier_perf
ORDER BY delay_rate ASC;
```

Metadata enabled carrier_code discovery.
